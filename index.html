cdf2fst

NetCDF to Standart RPN file converter

Download this project as a .zip file Download this project as a tar.gz file

This project is intended to provide support of the NetCDF format for GEM. NetCDF is a de facto standard format for output from various atmospheric models. However GEM cannot read or write this format directly. So, cdf2fst could be used to convert data from other model to supply it to GEM.
